    Stock Entry
    Stock Purchase

    This option is used to enter any bill entry received from the supplier. This page will show the list of category and schemes then after selecting the options the form for purchasing

    Stock Issue

    This option is used to enter any item issuing by the admin to the customer. This page is also inlist the consumable and equipment item. We can also see the list of the category and schemes then after selecting th9e options the form for issuing

    Add New
    Departments

    This option is used to add any new departments that doesn't exit earlier.This page also shows the list of all department exist in the database.

    Items

    This option is used to add any new item of any type that doesn't exit earlier.This page also ask to fill what type of item is and in what quantity. eg- mouse,keyboard,server,switch,laptop etc

    Schemes

    This option is used to add any new scheme eg- faculity, student.

    Vendor

    This option is used to add new Vendor also need to fill the GST number, address and item purpose of the supplier. this page also shows all previous supplier name that had been stored in the databases.

    View

    This page shows the list of all Data Item, Purchased Item,Issued Item.You can see the result by selecting the category. Also you can see result in sorting order of your choice.

    Help

    This option is used to take a look how all the modules are working .

    Logout

    This option is used to logout from CCIMS and redirect into the home page.
